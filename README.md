GirlyNote is a Markup Language made as an alternative to Markdown.

# Syntax
| Input       | Usage                           |
|:-------     |:---------------                 |
| `~# Text #~`| Makes a header                  |
| `~# Text`   | Makes a comment                 |
| `~#text~`   | Makes a tag                     |
| `~ Text`    | Makes an unordered bullet point |
| `~n Text`   | Makes an ordered bullet point   |
| `~\| Text : Text \|`| Makes a table           |
